# Design System

## Concept

Minimal + Retro
Modern UI with subtle skeuomorphic feel

## Colors

Background: #F5F1EA
Primary: #F28C28
Text: #3A2E2A
Secondary: #E8E1D9

## Buttons

- Rounded (16~24px)
- Soft shadow
- Press scale: 0.92
- Slight depth

## Cassette

- Clean, not overly realistic
- Focus on reels animation
- Balanced visual weight

## Animation

- Smooth
- Physical feeling
- No flashy motion

## Flip Button

- Separate from controls
- Clear CTA
- Slight distance from main controls

## Noise UI

- Slider (0~5s)
- Subtle, not dominant

## DO NOT

- Neon colors
- Glassmorphism
- Heavy gradients
- Game-like UI
